# 683280yj.github.io

姚军的个人主页，部署于 GitHub Pages。

访问地址：[https://683280yj.github.io](https://683280yj.github.io)

## 技术栈

- HTML5
- CSS3（CSS 变量、Grid、Flexbox、动画）
- Vanilla JavaScript（Intersection Observer、平滑滚动）
- GitHub Pages + GitHub Actions 自动部署

## 本地预览

直接在浏览器中打开 `index.html`，或使用任意静态服务器：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
