三创比赛小计

2024-07-23

第十四届全国大学生电子商务“创新、创意及创业”挑战赛（简称三创赛）的产学用（buc）实战赛，是竞赛组织委员会联合武汉惠赠你网络科技有限公司设立的。该赛道的相关内容如下：

比赛背景：为了深入贯彻落实《“十四五”教育发展规划》和《国务院办公厅关于深化产教融合的若干意见》，深化产教融合，促进教育链、人才链与产业链、创新链有机衔接，以推进人力资源供给侧结构性改革，在提高教育质量、扩大就业创业、推动经济转型升级、培育经济发展新动能等方面具有重要意义。

比赛目的：

  1. 为大学生提供零成本、低风险的实习实践、就业创业机会，使他们在毕业前积累实践技能和创业经验，减轻家庭经济负担。
  2. 利用大学生营销队伍，为企业低成本拓展销售渠道，企业和大学生在销售成功后均可获利，失败则无损失。
  3. 响应国家政策，解决大学生就业压力，整合高校、企业资源，为大学生就业创业打下基础，平台还可为优秀的大学生促销员提供就业岗位。
  4. 促进高校和企业之间的相互合作，将大学整合到供应链中，实现企业、大学、消费者的有机统一，以及“政产学研用”的密切结合。
  5. 为消费者提供便利、个性化的服务，大学生作为服务主体具有较高信誉度和认可度，且能为消费者提供个性化服务并优化其购物环节。

比赛内容：武汉惠赠你网络科技有限公司是本届产学用（buc）实战赛唯一指定合作企业，提供技术及运营保障支持。参赛团队通过产学用（buc）实战平台，与国内主流企业联合，以校园 o2o 模式、同城本地生活及海量源头供应链一件代发模式为组合开展实战。该平台推出“千企陪跑计划”，遴选千位企业家与参赛团队结对陪跑，并在每个赛区搭建本地化的产学用赛事运营服务中心，为参赛团队提供供应链产品支持、实战指导培训、营销资源对接、实习就业、成果转化等全方位服务。

报名流程：

  1. 参赛团队通过三创赛官网报名，选择实战赛中的产学用（buc）实战赛，并填写具体报名信息。
  2. 团队提交报名信息后，队长添加赛事服务工作人员企业微信，工作人员将在1个工作日内发送参赛手册至队长微信，并在3个工作日内授予参赛团队《三创赛产学用（buc）实战赛团队》参赛证。
  3. 参赛团队根据赛事培训手册指引和工作人员协助，完成相关产学用（buc）实战平台账号的绑定，即可进入实战落地环节，正式开始“产学用（buc）”实战运营。
  4. 各高校校赛组委会对报名后的参赛团队提交的报名信息进行审核，为通过审核的参赛团队提供学校支持，以便“产学用（buc）”平台在其后对该团队进行准确的指导和服务。

实践成长：

  1. 第十四届三创赛已于2023年9月21日正式开始报名，团队报名后即可参加产学用（buc）实战赛并进行项目运营。校赛、省级赛、国赛的实战赛数据统计时间均为赛前一周截止。
  2. 参赛团队将与“产学用（buc）实战赛”平台的本赛区赛事运营服务中心衔接，学习赛事培训手册和操作方法，由赛事运营服务中心指导搭建部署实战店铺，并在平台上进行产品营销。
  3. 产学用（buc）实战赛仍然按校赛、省级赛、国赛三级赛事比赛，校赛、省级赛与相应的常规赛一起举行，国赛独立举行。参赛团队现场参加各级比赛时需向评委提供《项目报告书》（纸质版、《参赛团队承诺与说明书》附在封二）、演讲文稿（ppt 电子版）。小组赛参赛团队答辩时间共15分钟，其中团队演讲8分钟（≤8分钟），评委提问与参赛团队回答7分钟（=7分钟）；终极赛每个参赛团队做8分钟（≤8分钟）演讲。

评分规则：第十四届三创赛竞赛规则规定，在实战赛环节比赛评分细则中创业项目分值为45分，团队在实战赛中的创业业绩将作为三创赛文档中创业部分的内容。

plog（Photo Blog）与一般的视频、写文章、写公众号以及拍抖音在以下方面可能存在区别：

  1. 形式：plog 主要以照片为主要展示形式，通常会搭配简短的文字说明；视频则是以动态影像为主；文章和公众号更侧重于文字表达；抖音则以短视频为特色。
  2. 内容重点：plog 更注重通过照片来记录和分享生活中的瞬间、故事或情感；视频可以更全面地展示场景和动态过程；文章可以深入阐述观点和讲述故事；公众号可以涵盖各种类型的内容；抖音则常常以娱乐、创意或短视频内容为主。
  3. 制作难度：相对来说，plog 的制作可能相对简单，只需要拍摄照片并进行简单的编辑和排版；视频的制作可能需要更多的拍摄和后期编辑工作；写文章需要有较好的文字表达能力；拍抖音则需要有创意和表演能力。
  4. 传播方式：plog 可以在社交媒体上分享，吸引关注和互动；视频在视频平台上有更广泛的传播渠道；文章和公众号可以通过订阅和分享来传播；抖音则主要在抖音平台上传播，具有较强的社交互动性。
